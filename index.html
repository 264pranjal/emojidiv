Emoji DIV

A Hacktoberfest Project of Women Who Code Manila

GUIDELINES:

  1. Fork this repository.
  2. Create a div using your name as the ID.
  3. Select an emoji to make.
  4. Write a single div representation of your emoji.
  5. Push your work and create a pull request.
  6. Enjoy!
Copy the div with class emoji and start drawing your chosen emoji with CSS! Don't forget to add your name!
